Неделя 1-2: Введение в Автоматизацию Тестирования и Основы Python

Урок 1: Введение в Автоматизацию Тестирования

  • Что такое тестирование? Введение в основные понятия: баг, дефект, жизненный цикл дефекта.
  • Виды тестирования: функциональное, нефункциональное, регрессионное, дымовое, санитарное.
  • Автоматизация тестирования: цели, преимущества, недостатки.
  • Основные этапы жизненного цикла ПО и роль автотестировщика.
  • Обзор основных процессов разработки: Agile, Scrum, Kanban, Waterfall. Роль тестировщика в каждом процессе.
Практика:
  • Установка и настройка Python.
  • Установка IDE (например, PyCharm, VSCode).
  • Создание простого Python-скрипта для вывода "Hello, World!".
Примеры:
  • Пример простого ручного тест-кейса и его перевод в автоматизированный тест.

Урок 2: Основы Программирования на Python для Тестировщиков

  • Основные конструкции Python: переменные, типы данных, условия, циклы.
  • Функции и модули: создание и использование.
  • Работа с файлами: чтение и запись данных.
  • Введение в обработку исключений и логирование.
Практика:
  • Написание скриптов для выполнения базовых операций (например, вычисление факториала, работа со строками).
  • Создание функций для повторяющихся операций.
  • Добавление логирования в скрипты для отслеживания выполнения операций.
Примеры:
  • Скрипт для чтения данных из текстового файла и их обработки.
  • Логирование шагов выполнения скрипта.
